OSC Forwarder

This app will connect to a socket.io server (at the address you input below), listen for socket messages with the title 'osc', and rebroadcast them to an OSC server on the local network as osc messages. It will also create an osc server on the local network, and forward OSC messages received by that server to the socket server.
